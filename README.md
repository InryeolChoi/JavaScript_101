* 자바스크립트의 문법(반복문, 조건문, 자료구조 등등..)을 활용하는 것도 중요한 목표 중 하나.
* javascript 런타임인 node.js의 여러가지 기능을 연습.
* 여러가지 모듈 활용해보기 + 간단한 웹 서버를 구축.
* 향후 진행할 express, next의 기반.
